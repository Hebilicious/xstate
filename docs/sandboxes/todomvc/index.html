# TodoMVC App Template

Template used for creating TodoMVC (opens new window) apps

# Getting started

  • Read the Application Specification (opens new window) before touching the template.

  • Delete this file and rename app-readme.md to readme.md and fill it out.

  • Clone this repo and install the dependencies with npm (opens new window) by running: npm install.
